# 🐾 VetCare - Aplicación Web Veterinaria

Aplicación web desarrollada con **React.js** que permite gestionar servicios veterinarios, visualizar mascotas y agendar citas de forma sencilla.

## 🚀 Tecnologías utilizadas

* React.js
* CSS
* React Router
* React Icons
* Consumo de API pública para imágenes de mascotas
* JavaScript (ES6)

## 📋 Funcionalidades

* 🏠 Página de inicio con información de la veterinaria
* 🐶 Visualización de mascotas (perros y gatos) usando API pública
* 🩺 Sección de servicios veterinarios interactiva
* 📅 Formulario para agendar citas
* 🔔 Mensajes de confirmación al realizar acciones
* 🎨 Interfaz moderna, interactiva y responsiva
* 🧭 Navegación entre páginas con React Router

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio

git clone URL_DEL_REPOSITORIO

2. Instalar dependencias

npm install

3. Ejecutar la aplicación

npm run dev
